# 🚀 AI Automation Workflow using n8n

## 📌 Project Overview
This project demonstrates an **AI Automation Workflow** built using **n8n**.  
The workflow processes incoming webhook data, applies conditional logic, integrates with an external API, and returns a structured JSON response.

---

## ⚙️ Workflow Architecture

Webhook → IF → HTTP Request → Code → Respond to Webhook

---

## 🔧 Features

- 📥 Accepts input via Webhook (GET request)
- 🔍 Applies logic using IF node
- 🌐 Calls external API using HTTP Request node
- 🧠 Processes data using JavaScript (Code node)
- 📤 Sends structured JSON response using Respond to Webhook node

---

## 🛠️ Technologies Used

- **n8n** (Workflow Automation Tool)
- **JavaScript** (Code Node)
- **REST API** (HTTP Request Node)
